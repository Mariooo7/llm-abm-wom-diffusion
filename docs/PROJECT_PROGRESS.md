## 📅 里程碑 3：终局架构与机制定型 (2026-03-19 至 03-20)

### 1. WOM 分桶策略升级 (环境概率分布)
- **触发问题**：早期版本中，LLM 对静态数值 `emotion_arousal` 过于敏感，导致强组易出现“天花板效应”（秒满），弱组易出现“地板效应”（贴底），网络结构的观测窗口被压缩。
- **解决方案**：
  - 剔除静态数值输入，采用“环境概率分布抽样”机制。
  - 新增 `wom.high_arousal_ratio`，强组设定为 0.6，弱组设定为 0.3。
  - 核心逻辑：让 LLM 纯靠阅读历史口碑文本感知情绪，而非响应数值指令。
- **理论支撑**：信息环境分布（而非单条极端消息）作为操纵对象，更符合传播学中对情绪信息占比的处理路径（如 Berger & Milkman, 2012）。

### 2. 初始火种与自发创新解耦
- **触发问题**：宏观 Bass 模型的 $p$ 值在微观 ABM 离散化时，导致“冷启动失败（0 采纳）”与“后续创新概率”相互纠缠。
- **解决方案**：
  - 物理剥离初始火种机制。引入 `initial_seed_ratio=0.04` 确保所有网络拓扑开局均拥有等量的初始扩散源。
  - 维持 $p=0.003$ 作为后续极低概率的自然采纳基线。

### 3. 工程大一统与纯净化
- 废弃早期多分支的 `run_pilot.sh` 和 `run_batch.sh`，统一重构为单一入口 `scripts/run_experiment.sh`。
- 全面清理冗余文件，完善 `.gitignore` 规则，确保远程仓库仅保留最纯净的代码与学术配置，符合开源发布规范。

## 📅 里程碑 2：参数校准与结构效应捕捉 (2026-03-11 至 03-17)

### 1. p/q 参数的“跷跷板”校准
- **初始预设**：依据经典文献设定基线，但在 LLM Agent 体系下出现了严重的不适应（强组过强，弱组过弱）。
- **校准路径**：
  - *压制天花板*：大幅下调强组相关参数，避免小世界网络的局部优势被全局极化信息淹没。
  - *拯救冷启动*：针对弱组多次微调 $p$ 与 $q$，确保其在仿真周期内不会全军覆没。
- **经验总结**：大模型对文本的响应存在非线性阈值，必须通过控制 `q`（模仿系数）来调节同侪压力，以平衡纯文本驱动下的保守倾向。

### 2. 并发调度与稳定性保障
- **Go 网关升级**：将可重试错误判定统一收敛至 `is_retriable_decision_error_message`。
- **局部重试机制**：在 Agent 的 `step` 中引入决策级重试（失败先局部重试，耗尽再触发 run 级重试），极大提升了正式长周期实验的成功率。
- **性能探测**：确立了 `REPETITION_WORKERS=4` 为当前资源环境下的稳定并发极值。

## 📅 里程碑 1：核心机制奠基 (2026-03 上旬)

### 1. 混合架构确立
- 确立 `Go (高并发调度) + Python (Mesa 引擎)` 的双生架构。
- 坚持“研究语义优先”原则：工程优化仅在 `Repetition` 级别做并发，单次 Run 内严格保持随机异步更新（Asynchronous Activation），确保因果推断的严谨性。

### 2. 离线 WOM 语料库机制
- 放弃“运行时动态生成文本”，改用离线 LLM 生成并人工质检的固定语料库（`wom_corpus.csv`）。
- **意义**：将 WOM 转化为标准的实验刺激材料（Stimulus），保证了实验的可控性与完全可复现性。

## 📚 关键参考文献支撑

在校准与机制设计中，本研究主要参考了以下核心文献：
1. **Bass (1969)**: A new product growth for model consumer durables. (*为 $p$ 与 $q$ 的参数分离与语义提供基础*)
2. **Watts & Strogatz (1998)**: Collective dynamics of 'small-world' networks. (*小世界网络聚集系数与平均路径长度的理论源头*)
3. **Centola (2010)**: The spread of behavior in an online social network experiment. (*支持“弱产品在小世界中更易通过复杂传染存活”的假设*)
4. **Berger & Milkman (2012)**: What makes online content viral? (*支持 High Arousal 情绪唤醒度作为信息极化分类的标准*)
